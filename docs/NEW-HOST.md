# Adding a New Host

## 1. Create Host Directory

```bash
mkdir -p hosts/<hostname>
```

## 2. Add Host Files

Create the following files under `hosts/<hostname>/`:

- `hardware-configuration.nix` — generated by NixOS installer, copy from `/etc/nixos/hardware-configuration.nix`
- `configuration.nix` — system config, use `hosts/satella/configuration.nix` as reference
- `home.nix` — home-manager config, use `hosts/satella/home.nix` as reference
- `packages.nix` — system packages and fonts, use `hosts/satella/packages.nix` as reference

## 3. Add to Flake

Add the host under `nixosConfigurations` in `flake.nix`:

```nix
nixosConfigurations.<hostname> = nixpkgs.lib.nixosSystem {
  system = "x86_64-linux"; # or aarch64-linux, aarch64-darwin, x86_64-darwin

  specialArgs = {
    inherit inputs;
  };

  modules = [
    ./hosts/<hostname>/configuration.nix
    # ...other modules
  ];
};
```

## 4. Add to Hosts Table

Add the new host to `hosts/README.md`.

## 5. Create Secrets Directory

```bash
mkdir -p secrets/<hostname>/<user>
```

## 6. Add Age Key to `.sops.yaml`

Generate a new age key on the new host:

```bash
age-keygen -o ~/.config/sops/age/keys.txt
```

Add the public key to `.sops.yaml`:

```yaml
keys:
  - &<hostname>_<username> age1...

creation_rules:
  - path_regex: secrets/<hostname>/.*\.(yaml|json|ini|env|bin)$
    key_groups:
      - age:
          - *<hostname>_<username>
```

## 7. Build and Switch

```bash
nixos-rebuild switch --flake ~/dotfiles#<hostname> --sudo
```
